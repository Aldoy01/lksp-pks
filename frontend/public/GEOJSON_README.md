# Indonesia GeoJSON

Unduh file GeoJSON Indonesia dan simpan sebagai:
  frontend/public/indonesia-provinces.geojson

Sumber yang direkomendasikan:
  https://github.com/superpikar/indonesia-geojson

Format yang diharapkan:
  - properties.kode  atau  properties.KODE_PROV  → kode BPS provinsi (e.g. "11", "31", "94")
  - properties.nama  atau  properties.Propinsi   → nama provinsi

Tanpa GeoJSON, peta akan tampil dengan circle marker sebagai fallback.
